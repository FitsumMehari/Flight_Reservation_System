How to Create an Account

First, Go to Landing Page and click “Get Started”.

Second: Input your name, Email address and password

Third: Create account.

If you already have an account, all you have to do is Sign In.

How to Reserve a Flight
  • Sign in and select your preferred flight from the available flights
  • Click on reserve a flight and input your information
  • Voila!, you have reserved a flight instantly.
